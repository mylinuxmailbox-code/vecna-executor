# ProcessExplorer

A Windows Forms application with animations, Lua scripting, and a pixel buddy character.

## Features
- **Process Management**: View all running processes with memory usage
- **Lua Scripting**: Write and execute Lua scripts (educational interpreter)
- **Animations**: Fade-in forms, button hover effects, staggered list loading
- **Toast Notifications**: Slide-in alerts from the right
- **Pixel Buddy**: Animated character that walks, types, celebrates, and panics
